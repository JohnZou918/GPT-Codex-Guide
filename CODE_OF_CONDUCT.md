# 社区行为准则

## 我们的承诺

GPT-Codex-Guide 致力于建立友好、尊重、开放合作的学习社区。无论年龄、经验、身份、背景或技术水平如何，每位参与者都应获得平等和有尊严的对待。

## 期待的行为

- 使用友好、耐心和包容的语言。
- 尊重不同经验、观点和学习速度。
- 对内容提出具体、可执行的建议，不攻击个人。
- 承认错误，愿意修正，并感谢他人的帮助。
- 优先保护零基础读者的安全、隐私和学习体验。
- 在分歧中围绕事实、验证方法和项目目标合作。

## 不可接受的行为

- 侮辱、歧视、骚扰、威胁或人身攻击。
- 发布他人的私人信息或敏感信息。
- 故意破坏讨论或提交误导内容。
- 利用经验差距嘲笑、羞辱或操纵新手。
- 其他不适合专业、友好社区的行为。

## 适用范围

本准则适用于仓库、Issue（问题讨论）、Pull Request（修改审查）、评论及代表本项目参与的其他公开交流。

## 处理方式

发现不当行为时，请通过 GitHub 仓库维护者提供的私密联系方式报告，并附必要上下文。维护者会尽量保护报告者隐私，并根据影响程度采取提醒、删除内容、限制参与或禁止参与等措施。

对善意错误优先说明和修正；对骚扰、威胁、泄露隐私或持续恶意行为采取更严格措施。

## 共同目标

目标不是让所有人意见相同，而是让不同经验的人可以安全地提问、学习、贡献并共同改进本指南。
