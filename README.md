# GPT-Codex-Guide

一本面向完全零基础用户的中文学习指南，帮助你理解并使用 ChatGPT、Codex、GitHub、Git、VS Code、Windows 和 macOS。

> 当前为第一阶段：已建立项目结构、首页和各章大纲。详细教程、截图与练习将在后续阶段补充。

## 1. 为什么学习 ChatGPT 与 Codex

ChatGPT 适合解释概念、讨论想法和整理需求；Codex 适合进入真实项目，阅读文件、执行修改并检查结果。学会两者的分工，你就能把“我想做什么”逐步变成“电脑上真正完成了什么”。

本书不会默认你理解命令行、仓库或提交。每个专业词第一次出现时都要解释，每个操作步骤都要说明为什么这样做。

## 4. 本书适合哪些人

- 从未接触编程、Git 或命令行的人。
- 想用 ChatGPT 和 Codex 做真实项目，却不知道工具如何连接的人。
- 希望每一步都解释原因、结果和出错检查方法的人。
- 同时使用 Windows 与 Mac，希望项目保持一致的人。
- 想建立固定开发流程的个人学习者。

## 如何使用本书

1. 先理解“这是什么”和“为什么存在”，再开始安装。
2. 按编号操作，不跳过每一步的原因说明。
3. 把练习放入 examples，把复用材料放入 templates。
4. 遇到问题先保留完整提示，再查看第 12 章。
5. 每次收工前使用第 13 章检查保存与同步状态。
